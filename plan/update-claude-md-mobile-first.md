在 CLAUDE.md 中補充手機優先設計 (Mobile-First Design) 的相關資訊，幫助未來的開發者理解專案的響應式設計理念。

## 規劃內容

### 1. 新增章節位置

在「重要技術細節」章節後，新增「響應式設計原則」章節。

### 2. 章節內容結構

**響應式設計原則**：
- **手機優先設計 (Mobile-First)**：
  - 說明基礎樣式以手機裝置為預設
  - 使用 `min-width` media query 漸進增強
  - 主要斷點：520px

- **關鍵響應式元素**：
  - 座標圖尺寸調整（手機 280×280px，桌面 400×400px）
  - 按鈕尺寸與間距
  - 文字大小與行高
  - 容器內距 (padding)

- **觸控優化**：
  - 按鈕點擊區域至少 44×44px
  - 充足的元素間距，避免誤觸

- **效能考量**：
  - CSS 使用 `transform` 做動畫，避免觸發 layout reflow
  - 圖片延遲載入（如需要）

### 3. 程式碼範例

提供一個簡單的 media query 範例，說明如何從手機樣式擴展到桌面樣式。
